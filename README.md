# 📖 Sobre o Projeto

Este projeto consiste em uma **landing page inspirada no layout da Escola DNC**, desenvolvida com o objetivo de praticar conceitos fundamentais de **desenvolvimento web front-end**.

A página apresenta informações sobre cursos, conteúdo educacional e inclui um **formulário para captura de dados dos usuários**, simulando páginas utilizadas para geração de leads em sites educacionais.

---

# 🚀 Funcionalidades

✔ Layout moderno baseado em landing pages  
✔ Estrutura HTML semântica  
✔ Estilização com CSS  
✔ Interatividade com JavaScript  
✔ Formulário de cadastro de usuários  
✔ Estrutura preparada para captura de dados
